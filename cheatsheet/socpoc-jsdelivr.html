CDN Gratuit

JSDelivr Cheat Sheet

Guide complet pour intégrer JSDelivr a vos sites statiques. CDN gratuit, rapide et fiable pour npm et GitHub.

Demarrage Rapide

Format de base pour charger n'importe quel package npm :

<!-- CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/PACKAGE@VERSION/FILE.css">

<!-- JavaScript -->
<script src="https://cdn.jsdelivr.net/npm/PACKAGE@VERSION/FILE.js"></script>

Packages npm

Tailwind CSS

tailwindcss 
<script src="https://cdn.jsdelivr.net/npm/tailwindcss@3.4.1/lib/index.js"></script>

Alpine.js

alpinejs 
<script defer src="https://cdn.jsdelivr.net/npm/alpinejs@3.x.x/dist/cdn.min.js"></script>

Three.js

three 
<script src="https://cdn.jsdelivr.net/npm/three@0.160.0/build/three.min.js"></script>

<!-- Avec OrbitControls -->
<script src="https://cdn.jsdelivr.net/npm/three@0.160.0/examples/js/controls/OrbitControls.js"></script>

GSAP

gsap 
<!-- Core seulement -->
<script src="https://cdn.jsdelivr.net/npm/gsap@3.12.4/dist/gsap.min.js"></script>

<!-- Avec ScrollTrigger -->
<script src="https://cdn.jsdelivr.net/npm/gsap@3.12.4/dist/ScrollTrigger.min.js"></script>

Anime.js

animejs 
<script src="https://cdn.jsdelivr.net/npm/animejs@3.2.1/lib/anime.min.js"></script>

Fichiers GitHub

Charger n'importe quel fichier depuis un depot GitHub :

<!-- Format -->
<script src="https://cdn.jsdelivr.net/gh/USER/REPO@VERSION/FILE.js"></script>

<!-- Exemple: fichier specifique -->
<script src="https://cdn.jsdelivr.net/gh/jquery/jquery@3.7.1/dist/jquery.min.js"></script>

<!-- Exemple: branche main -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/USER/REPO@main/style.css">

Naviguer dans un depot

Visualisez l'arborescence complete d'un package :

<!-- Ouvrir dans le navigateur -->
https://cdn.jsdelivr.net/npm/PACKAGE/

<!-- Exemple -->
https://cdn.jsdelivr.net/npm/three/

Gestion des Versions

Syntaxe Description Exemple
@version Version exacte @3.7.1
@^version Compatible (semver) @^3.7.1 → 3.x.x
@~version Patchs seulement @~3.7.1 → 3.7.x
@tag Tag specifique @latest, @next, @beta 
<!-- Derniere version 3.x.x -->
<script src="https://cdn.jsdelivr.net/npm/jquery@3/dist/jquery.min.js"></script>

<!-- Derniere version (risque de breaking changes) -->
<script src="https://cdn.jsdelivr.net/npm/jquery/dist/jquery.min.js"></script>

<!-- Version exacte (recommande pour production) -->
<script src="https://cdn.jsdelivr.net/npm/jquery@3.7.1/dist/jquery.min.js"></script>

Minification Auto

JSDelivr minimise automatiquement les fichiers si vous ajoutez .min :

<!-- Original -->
https://cdn.jsdelivr.net/npm/gsap/dist/gsap.js

<!-- Minifie automatiquement -->
https://cdn.jsdelivr.net/npm/gsap/dist/gsap.min.js

<!-- Forcer la minification -->
https://cdn.jsdelivr.net/npm/gsap/dist/gsap.js?output=min

ES Modules (esm.run)

Utilisez esm.run pour les imports ES6 natifs :

<!-- Dans le HTML -->
<script type="module">
  import gsap from 'https://esm.run/gsap';
  import * as THREE from 'https://esm.run/three';
  import { ScrollTrigger } from 'https://esm.run/gsap/ScrollTrigger';
  
  gsap.registerPlugin(ScrollTrigger);
</script>

<!-- Version specifique -->
import confetti from 'https://esm.run/canvas-confetti@1.6.0';

Template HTML Complet

Exemple d'un fichier HTML statique pret a l'emploi :

<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Mon Site</title>
  
  <!-- Tailwind CSS -->
  <script src="https://cdn.tailwindcss.com"></script>
  
  <!-- Google Fonts -->
  <link rel="preconnect" href="https://fonts.googleapis.com">
  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
  <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;600;700&display=swap" rel="stylesheet">
  
  <!-- Alpine.js -->
  <script defer src="https://cdn.jsdelivr.net/npm/alpinejs@3.x.x/dist/cdn.min.js"></script>
  
  <!-- Animate.css -->
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/animate.css@4.1.1/animate.min.css">
</head>
<body>
  <!-- GSAP + ScrollTrigger -->
  <script src="https://cdn.jsdelivr.net/combine/npm/gsap@3.12/dist/gsap.min.js,npm/gsap@3.12/dist/ScrollTrigger.min.js"></script>
  
  <script>
    gsap.registerPlugin(ScrollTrigger);
    // Vos animations ici
  </script>
</body>
</html>

Bonnes Pratiques

Performance

  • ✓ Utilisez SRI pour la securite
  • ✓ Prefetch les ressources critiques
  • ✓ Combinez les fichiers pour reduire les requetes

Stabilite

  • ✓ Fixez la version exacte en production
  • ✓ Testez avec @latest en developpement
  • ✓ Verifiez les releases sur npm/GitHub
